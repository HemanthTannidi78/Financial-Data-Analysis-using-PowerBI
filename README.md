# 📊 Financial Data Analysis Dashboard (Power BI)

## 🚀 Project Overview
This project presents a comprehensive **Financial Data Analysis Dashboard** built using **Power BI**, designed to analyze and visualize financial performance across multiple companies.

The dashboard provides insights into key financial metrics such as **Revenue, Net Income, Assets, Liabilities, and Cash Flows**, helping uncover trends and support data-driven decision-making.

---

## 🎯 Objectives
- Analyze financial performance across companies
- Compare revenue, profit, and cost structures
- Identify trends in operating, investing, and financing activities
- Build an interactive and user-friendly dashboard

---

## 📊 Key Insights
- 📈 Total Revenue exceeded **2 Trillion**, indicating strong market scale  
- 💰 Net Income reached **84.85 Billion**, with top companies driving most profits  
- 🏆 **Apple & Microsoft** are leading in net income generation  
- 🛒 **Walmart** has the highest revenue but also the highest cost of revenue  
- ⚖️ Total Assets (**1.66T**) vs Liabilities (**1.13T**) show overall financial stability  
- 📉 Operating income distribution highlights market concentration among top firms  
- 💸 Cash flow analysis:
  - Strong operating cash flow from tech giants  
  - Financing dominated by banking institutions  
  - Investing activities show variability indicating strategic decisions  

---

## 📌 Dashboard Features
- 📅 Year-wise interactive filtering  
- 📊 KPI cards for quick overview  
- 📉 Revenue vs Net Income comparison  
- 📦 Cost of Revenue analysis  
- 💵 Cash flow breakdown:
  - Operating Activities  
  - Investing Activities  
  - Financing Activities  
- 📈 Trend visualizations across companies
